珺和府 26-104 别墅空间漫游 V7 WebGL 优化版

使用方式：
1. 完整解压整个文件夹。
2. 建议通过本地服务打开，避免浏览器限制 WebGL 读取本地图片。
3. Mac 可在本目录运行：python3 -m http.server 8765，然后访问 http://127.0.0.1:8765/index.html
4. Windows 可继续使用“启动本地服务.bat”，或用任意静态服务器打开。

本版说明：
- 使用本地 WebGL 球面投影渲染 2:1 全景图，不依赖外部网站和外部 CDN。
- 支持左右环视、上下俯仰、滚轮视场缩放，热点随视角透视投影显示。
- 以已提取到的 3 处真实全景为基础：一层客餐厅、一层会客厅、夹层休闲区。
- 其余空间以楼层彩平和空间图集辅助展示。
- 当前素材只有 3 个真实全景节点，如需达到案例站那种完整漫游效果，需要继续补充更多空间节点并按真实动线标定热点。
